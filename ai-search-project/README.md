# AI Semantic Search using Endee

## Project Overview
This project demonstrates a simple AI semantic search system using the Endee Vector Database.  
Users can enter a search query, and the system returns the most relevant results using vector similarity.

## Technologies Used
- Python
- Endee Vector Database
- Sentence Transformers

## Project Files
app.py – Main application  
data.txt – Sample dataset  
requirements.txt – Python dependencies  
README.md – Documentation

## Installation
Clone the repository:

git clone https://github.com/Mathan-cse/endee.git

Go to the project folder:

cd ai-search-project

Install dependencies:

pip install -r requirements.txt

## Run the Project

python app.py

Enter a search query and the system will return the most relevant results.
